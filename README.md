# 🌐 AI Resume Analyzer - Frontend

## 🚀 Overview

This is the frontend of the **AI Resume Analyzer** project.
It allows users to upload their resume and analyze how well it matches a job description.

---

## ✨ Features

* 📂 Upload Resume (PDF)
* 📝 Enter Job Description
* 📊 View Match Score (with progress bar)
* ✅ Matched Skills
* ❌ Missing Skills
* 🎯 Role Detection
* 💡 AI-based Suggestions

---

## 🛠 Tech Stack

* React.js
* JavaScript (ES6)
* CSS (Inline Styling)
